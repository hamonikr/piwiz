## piwiz

하모니카 라즈베리파이용 arm64 버전에서

시스템 처음 부팅시 사용자 비밀번호 변경 및 환경 설정을 돕는 도구

 - 언어 및 시간대 설정
 - 비밀번호 변경
 - 기본 브라우저 설정
